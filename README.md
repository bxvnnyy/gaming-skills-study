# 🎮 Gaming & Cognitive Skills Study!

An interactive research project exploring the relationship between video game habits and cognitive performance in logic, spatial reasoning, and memory.

## 🚀 [Click Here to Participate in the Study](https://bxvnnyy.github.io/gaming-skills-study/)

### 📋 Project Overview
This study is designed to gather data on how different gaming genres (e.g., FPS, RPG, Strategy) correlate with specific mental "quests" 

**Sections include:**
* **Player Profile:** A survey of gaming habits and preferences.
* **Logic & Lore:** Quantitative reasoning puzzles.
* **The Map Room:** Verbal and spatial challenges.
* **The Prism Pattern:** A Simon-style memory test.

### 🔒 Privacy & Ethics
* **Anonymous:** No names, emails, or IP addresses are collected.
* **Consent:** Participants must be 13+ with parental permission.
* **Data Usage:** Data is stored in a private database and used solely for academic analysis.
