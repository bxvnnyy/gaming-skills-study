# 🎮 Gaming & Cognitive Skills Study

An interactive research project exploring how video game habits relate to cognitive performance across logic, verbal reasoning, spatial ability, and working memory.

---

## 🚀 TL;DR
You’ll complete:
- a short gaming habits survey 🎮  
- logic + verbal + spatial challenges 🧠  
- a fast-paced memory test 🟣  

Takes ~5–10 minutes. Fully anonymous. For a Year 12 research investigation.

👉 **[Click here to participate](https://bxvnnyy.github.io/gaming-skills-study/)**

---

## 📋 Study Overview
This project investigates whether different types of gaming experiences (e.g. FPS, RPG, strategy, casual games) are associated with differences in cognitive task performance.

Rather than testing “gaming skill,” the study focuses on *underlying cognitive processes* such as:
- logical reasoning under constraints
- verbal comprehension and interpretation
- spatial pattern recognition
- short-term memory and reaction accuracy

Participants complete a sequence of interactive tasks designed to measure these areas in a gamified format.

---

## 🧩 What You’ll Do

### 1. 🎮 Player Profile Survey
A short questionnaire about:
- gaming frequency and habits
- preferred genres
- device usage
- favourite games

### 2. 🧠 Logic & Reasoning Challenges
Problem-solving questions designed to measure structured thinking and deduction under time/choice constraints.

### 3. 📖 Verbal & Spatial Tasks
Tasks that involve:
- interpreting written information
- recognising patterns and relationships
- mental rotation / spatial reasoning challenges

### 4. 🟣 The Prism Pattern (Memory Test)
A Simon-style sequence memory task that tests:
- working memory capacity
- response accuracy
- pattern recall under increasing difficulty

---

## 🔒 Privacy & Ethics
- **Anonymous by design:** No names, emails, or personally identifying information are collected.
- **Participant IDs (optional):** Generated only for data organisation; not linked to identity.
- **Consent required:** Participants must be 13+ and have appropriate permission where required.
- **Data use:** Data is stored securely in Google Sheets and used solely for academic research purposes.
- **No tracking:** No cookies, analytics, or external tracking tools are used.

---

## 📊 Data Handling
All responses are:
- automatically recorded via Google Apps Script → Google Sheets
- stored in a structured dataset for later statistical analysis
- cleaned and anonymised before interpretation

The dataset will be used to explore correlations between gaming behaviour and cognitive performance, not to evaluate individuals.

## ⚠️ Important Notes
- This is a **non-clinical, educational research task**
- There are no right or wrong “real-world” abilities being measured
- Results are used for pattern analysis only, not diagnosis or grading
